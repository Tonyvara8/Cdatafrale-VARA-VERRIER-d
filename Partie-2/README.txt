Pour utiliser le CDATA-frame;
Télécharger et ouvrir le dossier dans un compilateur.
Utiliser le main.c .
  -> Plusieurs possibilités.
  -> Pour créer un CDATA-frame, taper 1.
  -> Pour créer et remplir une nouvelle colonne, taper 2.
  -> Pour afficher les colonnes, taper 3.
  -> Pour insérer une valeur dans une colonne, taper 4.
  -> Pour supprimer une valeur d'une colonne, taper 5.
  -> Pour supprimer une colonne, taper 6.
  -> Pour trier une colonne, taper 7.
  -> Pour chercher le nombre d'occurences d'une valeur dans une colonne, taper 8.
Les instructions individuelles de chaque fonctions s'afficheront dans la console.
